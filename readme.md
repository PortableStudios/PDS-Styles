## Installation

This is just installing from the GitHub repo.

```shell
npm install PortableStudios/PDS-Styles
# or
yarn add PortableStudios/PDS-Styles
# or
pnpm add PortableStudios/PDS-Styles
```

## Development

1. Fork
2. Clone
3. `pnpm install`
4. `pnpm start`
